1. Netlify Drop (Mais Rápido e Recomendado)

Arraste e solte sua pasta - deploy em 30 segundos!

Ir para Netlify Drop

  2. Arraste a pasta inteira "joguinhocriança" para a área
  3. Pronto! Seu site estará online instantaneamente

2. GitHub Pages (Mais Profissional)

Integração completa com GitHub, totalmente gratuito.

Criar Repositório GitHub
  1. Nome do repositório: jogo-biblico-educativo
  2. Marque "Public" e clique em "Create repository"
  3. Envie seu código: git push origin main
  4. Em Settings > Pages, ative "Deploy from a branch"

3. Vercel (Mais Moderno)

Performance excelente, deploy automático.

Ir para Vercel
  1. Importe seu repositório GitHub
  2. Framework: "Other"
  3. Build command: deixe em branco
  4. Clique em "Deploy"

4. Surge.sh (Mais Simples)

Deploy via linha de comando, sem configuração.

  1. Abra o terminal na pasta do projeto
  2. Rode: npm install -g surge
  3. Rode: surge
  4. Responda as perguntas e pronto!
