# INSTRUÇÕES PARA BAIXAR AS IMAGENS DOS ANIMAIS

## PASSO A PASSO PARA BAIXAR AS IMAGENS:

### 1. Acesse as URLs das imagens:

1. **LEÃO**: https://www.rawpixel.com/image/3060356/free-photo-image-lion-animal-forest
2. **ELEFANTE**: https://www.rawpixel.com/image/324454/vintage-elephant-poster
3. **ZEBRA**: https://www.rawpixel.com/image/3819803/illustration-image-tree-art-pattern
4. **MACACO**: https://www.rawpixel.com/image/12187313/png-white-background-cartoon-illustration
5. **TARTARUGA**: https://www.rawpixel.com/image/12210432/photo-image-cloud-sky-ocean
6. **GIRAFAS**: https://www.rawpixel.com/image/2603925/free-illustration-png-lion-monkey-giraffe
7. **TIGRE**: https://www.rawpixel.com/image/12494278/wildlife-savanna-mammal-animal-generated-image-rawpixel

### 2. Como baixar cada imagem:

1. Clique na URL da imagem
2. Procure pelo botão **"Download"** ou **"Baixar"**
3. Escolha a opção **"Free Download"** (grátis)
4. Se necessário, crie uma conta gratuita no RawPixel
5. Baixe a imagem em formato JPG ou PNG

### 3. Como salvar as imagens:

1. Salve cada imagem na pasta: `img/animais/`
2. Use os seguintes nomes:
   - Leão: `leao.jpg`
   - Elefante: `elefante.jpg`
   - Zebra: `zebra.jpg`
   - Macaco: `macaco.jpg`
   - Tartaruga: `tartaruga.jpg`
   - Girafas: `girafa.jpg`
   - Tigre: `tigre.jpg`

### 4. Redimensionar as imagens (opcional):

- **Tamanho ideal**: 400x300px
- **Formato**: JPG (melhor para fotos)
- **Qualidade**: Média a Alta

### 5. Ferramentas para redimensionar:

- **Online**: https://www.resizeimage.net/
- **Windows**: Paint (salvar como)
- **Celular**: App "Redimensionar Imagem"

## VERIFICAÇÃO FINAL:

Após baixar todas as imagens, verifique se:

- [ ] Todas as 7 imagens estão na pasta `img/animais/`
- [ ] Os nomes estão corretos
- [ ] As imagens têm tamanho adequado (400x300px ideal)
- [ ] O arquivo `jogo-imagens-reais.html` funciona localmente
