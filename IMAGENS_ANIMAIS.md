## 📝 COMO USAR ESTAS IMAGENS:

1. **Download**: Clique em cada URL e salve as imagens na pasta `img/animais/`
2. **Nome**: Use nomes descritivos: `leao.jpg`, `elefante.jpg`, `zebra.jpg`, etc.
3. **Tamanho**: Redimensione para 400x300px (ideal para quebra-cabeça)
4. **Formato**: JPG ou PNG
5. **Local**: Salve na pasta `img/animais/` do projeto

---

## 🎮 IMPLEMENTAÇÃO NO JOGO:

Para implementar estas imagens no quebra-cabeça:

1. **Carregue as imagens** como background de cada peça
2. **Use background-position** para mostrar diferentes partes da imagem
3. **Mantenha o mesmo sistema** de 10 peças (5x2)
4. **Cada animal** deve ser um quebra-cabeça completo
