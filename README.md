# Quiz Bíblico Educativo - **VERSÃO PURA (Sem Puzzle)** ✅

## 🎮 **O QUE É**

**Quiz infantil** com **100+ perguntas bíblicas**:
- **14 oportunidades**
- **10 acertos = VITÓRIA** + versículos inspiradores
- Interface linda, responsiva, replay infinita
- **SEM quebra-cabeça** - 100% quiz limpo!

## 🚀 **COMO JOGAR**
1. Abra `index.html` (tela inicial bonita)
2. Clique **"Começar Quiz!"**
3. Responda perguntas da Bíblia
4. Ganhe **versículos** em acertos
5. **10 acertos** = Parabéns! 🎉

## 📁 **ARQUIVOS PRINCIPAIS**
- `index.html` - Landing animada
- `jogo.html` - Quiz principal
- `style.css` - Design infantil (sem puzzle CSS)
- `script-corrigido.js` - Lógica quiz + 100 perguntas + versos

## 🌐 **DEPLOY FÁCIL**
```
GitHub Pages / Netlify / Vercel
- Copie tudo
- Deploy root: index.html
- Funciona offline!
```

## 🔧 **CONFIGURAÇÕES (script-corrigido.js)**
